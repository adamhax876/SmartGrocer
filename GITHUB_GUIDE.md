# 🚀 شرح مبسط لرفع مشروعك على GitHub (أفضل طريقة للنسخ الاحتياطي)

رفع مشروعك على GitHub يشبه عمل "خزنة إلكترونية" تحفظ كل حرف كتبته في المشروع، وتسمح لك بالرجوع لأي نسخة قديمة في حال حدث خطأ.

إليك الخطوات بأبسط طريقة ممكنة (اعتبرها مثل عملية إنشاء إيميل جديد):

---

### أولاً: إنشاء مستودع (Repository) في موقع GitHub
1. اذهب إلى موقع [GitHub.com](https://github.com/) وقم بتسجيل الدخول بحسابك.
2. في أعلى الصفحة على اليمين، اضغط على علامة الفائض `+` واختر **New repository**.
3. في مربع **Repository name** (اسم المستودع)، اكتب اسم مشروعك (مثلاً: `SmartGrocer`).
4. تأكد أن الاختيار محدد على **Private** (خاص) لكي لا يسرق أحد الكود الخاص بك.
5. انزل لأسفل الصفحة واضغط باللون الأخضر على زر **Create repository**.
6. ستظهر لك صفحة جديدة بها روابط، اترك هذه الصفحة مفتوحة وسنحتاجها بعد قليل!

---

### ثانياً: تحميل برنامج Git (خطوة تُعمل مرة واحدة فقط)
إذا كان هذا أول مشروع ترفعه في حياتك، فيجب أن تتأكد من تحميل أداة تسمى `Git`.
- **على الويندوز:** اذهب إلى [هذا الرابط](https://git-scm.com/download/win) وحمل البرنامج (Next > Next للآخر ثم تثبيت).

---

### ثالثاً: تهيئة المشروع ورفعه من جهازك (بواسطة VS Code)
لأنك تستخدم الفيجوال ستوديو كود (VS Code)، العملية سهلة جداً:

1. **افتح الطرفية (Terminal):** من شريط القوائم العلوي في VS Code، اضغط على `Terminal` ثم `New Terminal`.
2. **عَرّف نفسك (أول مرة فقط):** لكي يعرف النظام من أنت، انسخ هذين السطرين الصقهما في الـ Terminal (غيّر الاسم والإيميل لاسمك وإيميلك):
   ```bash
   git config --global user.name "Your Name"
   git config --global user.email "your.email@example.com"
   ```

3. **ابدأ عملية الرفع الدائمة للمشروع:**
   انسخ هذه الأوامر بالترتيب، أمراً أمراً، والصق كل سطر واضغط Enter:

   **الخطوة أ (تحزيم الملفات):**
   ```bash
   git add .
   ```
   *هذا الأمر مثل وضع كل ملفات الموقع في "كرتونة" تمهيداً لرفعها. لا يعطي رسالة نجاح وهذا طبيعي.*

   **الخطوة ب (تسمية الكرتونة):**
   ```bash
   git commit -m "النسخة الأساسية وتعديل اللوجو وتنبيهات الأدمن"
   ```
   *هذا الأمر يلصق ورقة على الكرتونة بها عنوان أو وصف لتعرف متى تم رفع هذه النسخة.*

   **الخطوة ج (الربط بموقع GitHub):**
   - ارجع للصفحة المفتوحة في المتصفح التي أنشأناها في "أولاً".
   - ابحث في منتصف الصفحة عن قسم يسمى `…or push an existing repository from the command line`.
   - ستجد تحته 3 سطور، أهم سطر هو الأول ويبدأ بـ `git remote add origin https://github.com...`
   - انسخ هذا السطر والصقه في الـ Terminal واضغط Enter.

   **الخطوة د والأخيرة (الصعود للفضاء 🚀):**
   ```bash
   git push -u origin main
   ```
   *بمجرد ضغط Enter هنا، قد يفتح لك نافذة صغيرة تطلب منك تسجيل الدخول (Sign in with browser). وافق عليها وانتظر حتى ينتهي الرفع لـ 100%.*

---

🎉 **انتهينا!**
الآن، اذهب إلى صفحة الـ Repo في GitHub التي تركناها مفتوحة، واعمل تحديث للصفحة (Refresh). سترى كل ملفات مشروعك الرائعة أمام عينيك مخزنة بأمان تام!
